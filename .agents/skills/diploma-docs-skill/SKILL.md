---
name: diploma-docs-skill
description: Use this skill when writing or updating diploma-related documentation, explanations, reports, README sections, or interpretation text.
---

Purpose:
Help Codex write documentation that can later be reused in the diploma.

Writing style:
- academic but simple;
- clear and concrete;
- no overclaiming;
- no vague AI-generated filler;
- connect implementation to the thesis goal.

Always explain:
- what was implemented;
- why it matters;
- how it supports energy efficiency analysis;
- how it connects IoT and ML parts;
- limitations of simulation-based data.

Preferred structure:
1. Purpose
2. Implemented component
3. Data or method used
4. Output
5. Relevance to the diploma
6. Limitations
