# Project Brief

The project develops a virtual compressed-air plant model for energy efficiency analysis.

The system simulates telemetry from virtual sensors, publishes data through MQTT, stores it in InfluxDB, visualizes it in Grafana, and exports the dataset for machine learning.

The ML part uses telemetry features to predict SEC and identify inefficient operating periods.

Expected practical outputs:
- architecture diagram;
- variable table;
- virtual plant workflow;
- data schema and MQTT payload example;
- InfluxDB storage plan;
- Grafana dashboard plan;
- final ML dataset;
- baseline and main ML model;
- metrics table;
- Actual vs Predicted SEC plot;
- Feature Importance plot;
- short interpretation.
