# Acceptance Criteria

## IoT part is complete when:

- architecture diagram exists;
- variable table exists;
- virtual plant workflow exists;
- data schema exists;
- MQTT payload example exists;
- dashboard plan exists;
- simulator generates 8640 valid rows;
- one inefficient scenario is included.

## ML part is complete when:

- final ML dataset exists;
- target is SEC;
- total_power_consumption is not used as feature;
- chronological 80/20 split is used;
- Linear Regression baseline is implemented;
- Random Forest Regressor is implemented;
- MAE, RMSE, and R2 are calculated;
- Actual vs Predicted SEC plot exists;
- Feature Importance plot exists;
- short interpretation exists.
